  1. What is CSS?

    CSS or Cascading Style Sheets is a Scripting language the same as HTML but has different role since HTML provides the HTML structure while CSS controls the webpage styling.

  2. CSS is a computer language for laying out and structuring webpages(HTML or XML).

  3. CSS makes the front-end of a website shine and it creates a great user experience.

  4. Basic tags of CSS are, (Caption-side, Column-Rule-Color, Column-Rule-Style, Column-Rule-Width, Column-Span) base on their definiton about CSS.

  5. HTML and CSS are both one but what differentiate them is there roles since HTML will create the website or webpage while CSS is the one designing and styling it.
